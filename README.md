# Creating a User in ServiceNow

This guide outlines the steps to create a new user account in ServiceNow and configure their initial login settings.

---

## Prerequisites
- Access to a ServiceNow instance  
- Appropriate administrative permissions to create and manage users  

---

## Steps

### 1. Navigate to User Administration
1. In the left navigation panel, click the **All** tab.
2. In the search bar, type **User Administration**.
3. Select **Users** from the menu.

---

### 2. Create a New User
1. On the Users page, click the **New** button located at the top right.
2. Enter the userâ€™s details, including:
   - **User ID**
   - **First Name**
   - **Last Name**
   - **Email**
   - **Department**
   - Any other required fields per your organizational standards  
3. Click **Submit** to create the user record.

---

### 3. Configure User Password
1. Search for the newly created user by name using the search bar.
2. Select the user from the results.
3. In the upper right corner of the user record, click **Set Password**.
4. Generate a random password and save it in a secure location if needed.
5. Enable the **Password needs reset** option so the user will be prompted to create their own password at first login.

---

## Completion
You have successfully:
- Created a user in ServiceNow  
- Set an initial password  
- Enabled password reset for secure first-time login  

The user account is now ready for final onboarding or role assignment.
